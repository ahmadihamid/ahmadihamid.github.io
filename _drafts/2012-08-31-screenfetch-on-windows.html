[+] bikin shortcut dari cmd fetch

[+] ubah namanya jadi hamid
[+] klik kanan di shortcut tadi and pilih properties
[+] ubah bagian target jadi "C:\WINDOWS\system32\cmd.exe /k C:\WINDOWS\system32\CMDfetch.bat -d"

[+] nah sekarang panggil shortcut tadi pencet tombol start - run - ketik hamid - enter
