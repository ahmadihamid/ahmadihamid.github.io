before it become a complete year without  "raspi diary" let's have a small article here.


we'll use RasPi as Manga Downloader using manga downloader and access the downloaded file via local ftp,
this may useful for you who life with poor internet connection and have less time just like me,


so we can leave our Raspi downloading our fav manga, and read it latter with light speed connection,


GET set!!
[1] Download and extract manga downloader
[2] Install python2
[3] Set an ftp server by following this wiki
or the wiki according your Linux distribution, here I use arch linux arm
optionally we can bind downloaded manga folder to ftp server folder, add this line to fstab file
[*] /home/xgaptek/mangadonlot/manga /srv/ftp/manga/ none    bind    0       0
let's try download xxxHolic from mangafox, create xml file for "manga downloader"
<?xml version="1.0" ?>
<root>
 <MangaSeries>
  <name>xxxHOLiC</name>
  <HostSite>MangaFox</HostSite>
  <LastChapterDownloaded>Vol.18 Ch.210</LastChapterDownloaded>
  <downloadPath>/home/xgaptek/mangadonlot/manga/xxxholic</downloadPath>
 </MangaSeries>

</root>
for more xml example

let's run the script
 
then we can read our loved manga, less stress after  boring and exhausting daily work
