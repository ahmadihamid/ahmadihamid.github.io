Mencegah Virus dari FD ke Kompi:

1. Matikan Autorun...

Langkah :
Pergi ke Group Policy dengan cara Start->Run->gpedit.msc

Nah buka "Administrative Templates" pada "Computer Configuration" dan "User Configuration" lalu pergi menuju folder "System". Pada kedua Folder "System" tersebut terdapat Setting "Turn Off Autoplay"... Klik 2 kali dan jadikan "Enable" dan pada pilihan "Turn Of Autoplay On" pilih "All Drives"...





2. Kalo mau browse FD sebaiknya gunakan Windows Explorer dan lihat pada jendela sebelah kiri. Hal ini untuk mencegah keteledoran User kalo2 ada Virus yang memanfaatkan icon berbentuk Folder yang akan mengecoh. Kalo di sebelah kiri kan gak akan muncul...

Contohnya Virus TATI, dkk...

3. Selalu hati2 dah, dan selalu perhatikan data2 di FD anda... Apakah ada perubahan, misalnya ada folder yang hilang atau apalah... Atau ada keanehan2 yang lainnya...

4. Tambahan dari kalian...

Mencegah Virus dari Kompi Ke FD:

autorun.inf... Walah kalo denger nih nama pasti bakalan eneg banget dah, coz pasti kalo colok FD ke warnet nih "barang" selalu ada. Apalagi Conficker dengan Folder "RECYCLER"nya yang ngebetein banget...

Oke saya berpikir neh (kalo salah mohon dikoreksi)...
Jika kita ingin melawan "FILE" gunakan "FOLDER" atau jika kita ingin melawan "FOLDER" gunakan "FILE"... Pokoknya jangan melawan
"FILE" dengan "FILE" atau "FOLDER" dengan "FOLDER"...

Maksudnya apa seh?

1. Untuk mencegah agar "FILE autorun.inf" tidak masuk ke FD kita harus membuat "FOLDER autorun.inf"... Kalo mau praktek coba aja buat folder autorun.inf di FD lalu buat file autorun.inf, lalu kopi ke FD...



Masih ragu? Oke... Masih inget artikel saya tentang cara membuat folder CON, AUX, NUL...




Review sedikit... Gak tau kenapa nama2 diatas tidak bisa dipakai untuk dijadikan nama folder. Tanya kenapa?

Tapi ternyata folder dengan nama tersebut dapat kita buat dengan menggunakan Command Prompt...

Caranya...

Buka Command Promt lalu ketik

md\\.\\C:\con


Liat di drive C maka akan ada folder con... Coba anda hapus!



Begitu pula dengan folder aux dan nul...

Untuk cara menghapusnya anda bisa menggunakan perintah...


rd\\.\\C:\con


Intinya dengan mensisipkan folder con, aux dan nul kedalam folder autorun.inf
maka gak akan ada yang bisa menimpanya... Apalagi hapus... "Untuk saat ini loh, gak tau kedepannya "

Caranya adalah (misalkan FD anda drive F:)

Buka command prompt dan ketik perintah berikut...


md\\.\\F:\autorun.inf\con (Enter)
md\\.\\F:\autorun.inf\con\nul (Enter)
md\\.\\F:\autorun.inf\con nul\aux (Enter)


Wakakakakakakak... Dengan sarat folder autorun.inf dah dibuat...

Nah untuk menghadapi "FOLDER RECYCLER"nya conficker kita gunakan aja file yang kita kasih nama "RECYCLER" tanpa ekstensi...
Kenapa pake FILE? Karena kalo pake FOLDER dengan nama yang sama file di dalam RECYCLERnya conficker akan masuk ke folder yang kita buat... Ane lupa namanya apa...

Contoh kita buat folder dengan nama RECYCLER di FD dan siapkan juga folder RECYCLER yang isinya terserah kamu...
Nah kopi RECYCLER yang ada isinya ke RECYCLER yang gak ada isinya... (Ngertikan? Sory kalo ribet)

Maka file yang di dalam akan ikut masuk walaupun kita menggunakan jurus folder con, aux dan nul.

2. Proteksi dengan menggunakan folder...

Pernah pake SMADAV? Yang terbaru kalo anda install maka di FD anda akan ada folder dengan nama "Δ Smad-Lock Δ".
