---[ first tips for us is "backing up our Arch"
we all know there's a "blood" on "bleeding edge" words
SO. back up can always minimize the blood spills.

and one of my favorite tools for backing up is "rsync"
so if you don't have back up yet,
let's fire up "rsync" so we can broke up our system anytime!


you can use :
---]# rsync -vrpoglHDIt /your/system/mount/* /your/backup/folder
for backing up
and use this to restore :
---]# rsync -vrpoglHDIt /your/backup/folder/* /your/system/mount
(u can use man rsync to see option meaning)

...

gah, what a simple tips right?

---[now, the second one is "make our system a lil easier to backup and restore"

what a mean about "easier" is "smaller" or something like that

so this is what I did on my system,
I make a symlink for few folder that contain many data :
/var/cache/pacman/pkg
/home/xgaptek/Documents
/home/xgaptek/Downloads
/home/xgaptek/Music
/home/xgaptek/Pictures
/home/xgaptek/Videos

one by one :
---] ln -s /folder/above /to/another/partition
example :
---]# ln -s /mnt/784f59a9-43c3-4d7c-829d-297a10894c91/TuxApp/cache/pacman/pkg /var/cache/pacman

that's all big folder I can think.
