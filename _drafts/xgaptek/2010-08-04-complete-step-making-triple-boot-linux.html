[+] DvD installasi WINDOWS SEVEN ULTIMATE or yg murah
[+] DvD installasi backtrack 4 or UBUNTU

skema harddisk yang aku sarankan :
skema ini bisa kalian ubah sesuai selera.
[+] partisi untuk LINUX : 12GB
file system : extended 3

[+] partisi swap LINUX : besarnya 2 kali ukuran RAM kalian(untuk ram 512 ke bawah saja)
file system : swap
partisi swap ini di linux berguna sebagai virtual memori.

[+] partisi untuk WINDOWS XP : 8GB
file system : NTFS

[+] partisi untuk WINDOWS SEVEN : 13GB
file system : NTFS

[+] partisi untuk menyimpan data atau document : sisa GB!
partisi ini kedepannya akan digunakan oleh WINDOWS maupun LINUX oleh karena itu aku saranin kita bikin file system-nya FAT-32 karena LINUX kurang baik dalam mounting file system NTFS. walapun sekarang telah banyak perbaikan, tetap aja LINUX lebih kenal FAT-32 daripada NTFS!!
jadi kita punya minimal 5 partisi..

Kondisi-kondisi/PKMA (Possibilities of Kondition that Might be Happened)


[------KONDISI--PERTAMAX------]

[+] komputer kosong/cuma ada dos ckck..kasian deh lu

ini posisi paling asik sebenernya karena kita bisa memanage dengan sangat baik harddisk kita dan kalian bisa pilih install XP or backtrack terlebih dahulu.

okeh kita pilih install backtrack duluan yaa..
kita mulai acak-acak komputer!!

1. booting komputer dari DvD backtrack
2. ketik perintah startx [ENTER] buat masuk ke mode GUI
3. buka konsole/terminal
4. ketik perintah gparted untuk menjalankan GParted

5. bikin partisi untuk backtrack dengan mengklik tulisan unalocated kemudian klik new kemudian atur spesifikasi sesuai kesepakatan kita di atas. isi field :
New Size (MiB) : 12000
Creat as : primary partition
filesystem : ext3
Label : bt4



6.bikin partisi swap, klik new.
New Size (MiB) : 2000
Creat as : primary partition
filesystem : swap
Label : swap

7.bikin partisi XP
New Size (MiB) : 7700
Creat as : primary partition
filesystem : ntfs
Label : xp

8.bikin extended partition
Free Space Preciding (MiB) : 0
Creat as : extended partition
"extended partition" adalah primary partition yang dapat memuat banyak logical partition. kita bisa mecah "extended partition" jadi banyak partition lagi.

9.kita bikin partisi buat SEVEN di dalam "extended partition" dengan cara mengklik tulisan unalocated kemudian klik new kemudian atur spesifikasi :
New Size (MiB) : 13000
Creat as : logical partition
filesystem : ntfs
Label : 7

10.bikin partisi buat document dengan cara mengklik tulisan unalocated kemudian klik new kemudian atur spesifikasi :
Free Space Preciding (MiB) : 44
filesystem : fat32
Label : hamid ganteng

fiuuuuuuuuuuuuuuuuuuhh..
inilah hasil kerja kita sejauh ini :

lanjuttt....


11. klik aplly!



12. Klik file “install.sh” yang ada pada desktop.

13. Tunggu beberapa saat hingga popup muncul , klik “continue anyway” jika ada pesan error “language crashed”

14. Pilih Lokasi anda : “Jakarta”, kemudian klk forward

15 Pilih keyboard layout : USA, klik forward

16. Pilih “Manual” pada saat proses prepare disk space, kemudian klik forward

17. Arahkan dan edit partisi yang akan di install linux,
partisi /dev/sda1 untuk linux backtrack-nya dan /dev/sda2 untuk swap-nya.

18. Klik pada /dev/sda1 , Klik edit untuk mengedit partisi /dev/sda1
Isi pilihan use as dengan “ext 3 journaling file system”
ceklist “format the partition”
Isi mount point dengan pilihan “ / ”
Klik OK



19. Klik pada /dev/sda2 , Klik edit untuk mengedit partisi /dev/sda2
Isi pilihan use as dengan "swap area"
Klik OK

20. Setelah selesai edit partisi, selanjutnya klik forward pada proses prepare install kemudian klik install.tunggu hingga selesai....

21. Finished. Silahkan reboot komputer kalian.

22. silakan nikmati OS geratisan anda itu!!

23. setelah puas menikmati ayo kita install windos

24. boot komputer dari cd installasi XP

25. lakukan installasi pada partisi yang kita siapin untuk XP (jangan salah pilih yaa..ingat labelnya partisi-nya "XP")

26. ikuti ja step-stepnya, gampang qo. tapi kalo kalian masih bingung boleh tanya di sini atau tanya developernya di sono

27. nah!! ga usah lama-lama nikmatin XP-nya, kita langsung aja install windos SEVEN..

28. masih di dalam XP. sekarang kalian masukan DvD windos SEVEN dan jalanin setup-nya. kita tidak melakukan installasi dengan mem-boot langsung dari DvD agar SEVEN tau diri kalo ada OS yang telah lebih dahulu terinstal di komputer. pengalamanku ketika menginstal SEVEN dengan mem-boot langsung dari DvD, SEVEN akan menyapu bersih MBR (Master Boot Record) sehingga kita enggak bisa lagi masup ke XP.

29.klik install untuk menginstal SEVEN dan ikuti aja langkah-langkahnya, mudah qo, lebih mudah dari install XP yang sangat mudah (keunggulan mikocok *geleng-geleng*)
tapi kalo kalian enggak pede silakan pelajari INI

30. Fixing MBR
ini adalah langkah-langkah untuk mengembalikan backtrack ke MBR:
#windos geser dikit backtrack mau duduk!!

[1]boot komputer dari dvd backtrack tercinta
[2]buka konsole
[3]ketik 'grub' (tanpa petik) , maka grub 'prompt' akan muncul

[4]ketik 'find /boot/grub/stage1' untuk mencari dimana grub-loader berada
#di sini saya memperoleh hasil (hd0,2)

[5]ketik 'root (hd0,2)'
[6]ketik 'setup (hd0)'
[7]ketik 'quit'

sebelum kalian reboot komputer ada baiknya kalian cek dulu menu.lst untuk memastikan LINUX dan WINDOWS telah masuk ke dalam GRUB-LOADER.
[1] masih di konsole
[2] cat (hd0,2)/boot/grub/menu.lst
[3] coba cek ada/tidak menu seperti ini :
# This entry automatically added by the Debian installer for a non-linux OS
# on /dev/sda1
title Windows
root (hd0,9)
savedefault
makeactive
chainloader +1

[4] kalo enggak ada, kopas aja tapi sesuaikan (hd0,9) dengan partisi SEVEN harddisk kalian.
#masih bingung?

[5] jalanin perintah find /Windows
#ntar keluar (hd0,X) itulah partisi Windows-7 kalian.

[6] edit menu.lst dan save
#selesai deh..tinggal reboot..insyaALLAH menu GRUB-LOADER bakal nongol lagee..
