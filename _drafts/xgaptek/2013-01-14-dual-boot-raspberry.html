--[ 1 ]--
first we need to make a room for our second OS,
so just use gparted-resize-and-make a new partition,

--[ 2 ]--
then just copy root partition from our disk image to our new second partition
you may use kpartx to mount disk image (.img file) then rsync to copy all the file,
orrr..
just rsync if you have installed image on another SD Card,
# kpartx -av disk-image.img
# mount /dev/mapper/loop0p2 /mnt/part2
# mount /dev/mmcblk0p3 /mnt/secos
# rsync -vrpoglHDIt --progress /mnt/part2 /mnt/secos
(adjust mount point according to your own system)
ah, we can also use "dd" for copying root partition directly from .img file but I find my partition become "unknown"
 
 
--[ 3 ]--
ok, now the final touch,
we need to copy pwnpi kernel  (our second os kernel.img) to /boot partition and rename it,
