if ya want to set one in home, u may need :

[=] an old TV with big body one (like my tv lol)
[=] raspberry pi machine
[=] high speed SDHC (I use 32GB class 10)
[=] good power source for our lovely device (I use android phone charger)
[=] AV cable (what u use for DVD player, yellow one)
[=] or HDMI cable if ya have modern TV or monitor
[=] powered usb hub (I bought cheap one from lil store, only $4 lol)
[=] keyboard, mouse, speaker
[=] usb wireless adapter (if ya want to use remote)
