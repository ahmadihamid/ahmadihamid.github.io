sebelum kita ngoprek si DeepFreeze alangkah baiknya bila kita kenalan ma c DeepFreeze..

DeepFreeze adalah sebuah sofware yang berfungsi untuk membodoh2i user. untuk membodoh2i anda DeepFreeze akan membuat sebuah mesin virtual yang mirip dengan komputer kalian sebelum DeepFreeze diinstal dan DeepFreeze membuat kalian bekerja di mesin tidak nyata itu SEJAK KALIAN STARTING UP komputer kalian.

DeepFreeze melindungi dirinya dengan password yang DISIMPAN DI $persi*.sys (kurang lebih namanya gitu. lupa..) sehingga orang yang enggak tau passwordnya atau lupa password atau enggak bisa baca $persi*.sys bakalan dibikin pusing.

di perkenalan DeepFreeze tadi kita punya dua point penting untuk ngoprek DeepFreeze :



[---DeepFreeze menyimpan password di $persi*.sys di root harddisk
[---DeepFreeze dijalankan saat start up


berhubung aku enggak bisa baca $persi*.sys buat intip passwordnya jadi kita replace aja file $persi*.sys yang ada di komputer korban dengan $persi*.sys dari komputer kita yang passwordnya kita tau (atau tak berpassword)


nah!

kita udah dapet satu cara tuh!

REPLACE $persi*.sys

masalahnya file ini diprotek!!

terpaksa kita replacenya lewat OS laen!!


kita pake live-cd linux aja yaa..




[---boot komputer dari cd linux kalian
[---replace file $persi*.sys
[---reboot
[---kalian udah bisa masup ke menu konfigurasi DeepFreeze

kita udah tau deepfreeze dijalankan saat komputer start untuk itu kita cari driver yang menjalankannya dan gigit driver nya. (DeepFrz.sys yang ada di WINDOWS/system32/drivers)



sayangnya kita g bisa gedit DeepFrz.sys langsung karena akan sia2.

[---boot komputer dari cd linux kalian
[---buka terminal

[---$ cp null.sys DeepFrz.sys
