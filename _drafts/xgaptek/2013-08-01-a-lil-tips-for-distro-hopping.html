Assalamualaykum,

if you are distro hopping that really like testing many linux distribution just like me,

maybe you still keep two or three distro on your computer installed, for some reason.

mine, on my laptop :
[1] Ubuntu 12.04, Ubuntu is my "first" so somehow I can't throw it from my computer
[2] Gentoo, just curious 'bout this one, not use it daily after strugle install
[3] Arch, lovely arch
 
so, as we know we'll have 3 (/home{/Document,/Downloads,/Music,/Pictures})
then if we use each of thoose distro often, we'll get trouble with many files,
so, it's worth to use only one (/home{/Document,/Downloads,/Music,/Pictures}) for all distro.

here we'll use symlink to prevent that,
for an example :
---]ln -s /mnt/784f59a9-43c3-4d7c-829d-297a10894c91/Documents /home/

then just repeat it for other folder you desired,
for the detail just look at the pic.


don't forget to make that partition automounted.

yaps..that's all.
take care and enjoy!
