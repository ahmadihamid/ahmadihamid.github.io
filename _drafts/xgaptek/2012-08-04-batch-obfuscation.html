"how to encript batch program"

bagi batch programer - yang sedikit pelit karena merasa amat sangat capek-capek sekali saat membuat programnya dan enggak mau scriptnya diedit oleh orang lain - dapat menggunakan cara ini untuk sedikit mempersulit orang lain untuk melihat kode yang anda buat..
ini sebenarnya bukan encription tapi cuma obfuscation, concealment, abscodence, dsb

kayaknya enggak perlu penjelasan panjang lebar,
kalian pasti paham hanya dengan melihat step berikut :

@echo off
echo hell o world
pause


kita enkripsi menjadi :

set minum=echo
set teh=off
set kopi=hell
set di=o
set cafe=world
set xgaptek.co.cc=pause
set d=c
set i=l
set a=s

@%minum% %teh%
%d%%i%%a%
%minum% %kopi% %di% %cafe%

%xgaptek.co.cc%

seperti yang kalian lihat trik di atas sangat mudah dicrack,
untuk lebih mempersulit cracking anda bisa membuat definisi-definisi yang lebih rumit,

contoh di atas cuma contoh dasar aja,
bagi kalian yang ngerasa kurang puas dengan cara diatas mungkin dapat mencoba program batch encriptor ini >>> http://depositfiles.com/files/2es55r2qb
