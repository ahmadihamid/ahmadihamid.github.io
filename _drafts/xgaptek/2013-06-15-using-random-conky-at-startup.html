a few days ago this thing came to my mind,
since I loves to download many conky config,
I have many of 'em in my home folder,
maybe so all of ya,

so why we don't use 'em all,
then we need some script to pick them (in this case randomly) at start up,
then fire up them,
so ya all can have differ conky-config every times computer booted-up,


so this is my lil script,
here we need a file that contain path to all conky script let's call it "kumpul"



#!/bin/bash 
sleep 2
random="`cat /path/to/kumpul | sort --random-sort | head -n 1`"
/usr/bin/conky --config=$random
exit 0
