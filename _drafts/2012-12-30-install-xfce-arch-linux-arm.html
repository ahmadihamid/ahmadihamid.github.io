OK!
luckily someone out there give us a disk image of an Arch Linux Arm that specially for RPi
this sud make it softcore now,

so just click the link and download,
dont forget read how to create the SD disk,
but in case you forget it, I put it here,
The SD card will need to be 2GB or larger.

    Linux
  1. Replacing sdX with the location of the SD card, run (please be careful when use dd):
    dd bs=1M if=/path/to/archlinux-hf-2012-09-18.img of=/dev/sdX
    Windows
  1. Download and install Win32DiskImager
  2. Select the archlinux-hf-2012-09-18.img image file, select your SD card drive letter, and click Write
as you see the image we download before created from 2GiB, so if yours bigger there sud be unallocated space in our SD card,
we need to resize it, I recommend easy way, using gparted.


OK!
if the SD ready, boot it. boot it. BOOT IT!!

put the SD and power up your RPi,
if everything OK, you'll get nuthing but this black screen,



as we all know,
arch didn't come with DE (desktop env.) as default,
but there are xfce, lxde, etc on repo,
we'll grab xfce (coz xfce is a lightweight complete desktop) from repo now,

run this command :
"pacman -Syu"
this performs a full system upgrade,
but before that you may want to backup files in vfat partition (the first partition on our SD)
I find myself get something  wrong after upgrade and need to replace the file in this partition with the old one,

OK next!
"pacman -S xorg-server xorg-xinit xorg-utils xf86-video-fbdev"
for Install the X server and video driver

"pacman -S xfce4 xfce4-goodies" Install XFCE desktop and a few additional software,

umm
I forget something,
we haven't make regular account till now,
sooo
let's make one,
fire up "useradd username"
substitute user with our regular one and start GUI with "startxfce4"
